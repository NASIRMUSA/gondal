# Gondal Provision Store Landing Page

This is a responsive, modern landing page for Gondal Provision Store Nig Ltd.

## How to Run
Simply open the `index.html` file in any modern web browser.

## Features
- **Hero Section**: Welcoming headline and CTA.
- **About Section**: Brief business description.
- **Products**: Highlight of top selling items with cards.
- **Why Choose Us**: Key benefits of the business.
- **Contact**: Functional-looking contact form and WhatsApp integration.
- **Responsive Design**: Works on mobile, tablet, and desktop.

## Technologies
- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (Mobile menu, Smooth scroll)
- FontAwesome (Icons)
- Google Fonts (Outfit)

## Images
Product images are generated using AI. If an image fails to load, a placeholder will be displayed.
